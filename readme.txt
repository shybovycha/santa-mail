==========================================================================
1. Огляд файлів та директорій на диску
==========================================================================

  На ліцензійному диску ви знайдете:
  
  \Santa Mail     Робочий каталог. В ньому знаходяться всі файли, необхідні 
                  для запуску та роботи гри. Для встановлення гри, скопіюйте 
                  його на ваш жорсткий диск.

  \DirectX        Робоча версія DirectX. Сумісність з програмою гарантовано.
  \install.exe    Файл швидкої інсталяції гри.
  \sources.zip    Тексти програми.
  \IDE.zip        Середовище розробки з компілятором.



==========================================================================
2. З чого почати
==========================================================================

  До запуску програми, ви повинні встановити найновіші (або хоча б
  перевірені та робочі) версії аудіо- та відеодрайверів та DirectX 9.0c.
  Для встановлення гри на цільовий комп"ютер, скопіюйте каталог "\Santa Mail"
  на жорсткий диск. Для запуску гри, виконайте файл "Santa Mail.exe" з
  папки, в яку встановлено програму.

  Для швидкого встановлення програми, запустіть файл "install.exe", що знаходиться на
  ліцензійному диску з програмою, оберіть каталог, куди б ви хотіли встановити
  гру та натисніть "Встановити".


==========================================================================
3. Системні вимоги
==========================================================================

  Для запуску та правильної роботи програми на комп"ютері еористувача
  повинно бути встановлене таке програмне забезпечення:
  1. DirectX 9.0 c (або вище). Увага: дана версія програми не 
    підтримує DirectX 10!
  2. Newton Game Dynamics Runtime (постачається в комплекті)
  3. Аудіо- та відеодрайвери
